# CMIP7-support

This repository serves as a central hub for requesting access to private CMIP7 and CDDS related repositories and for reporting general CMIP7 user support issues. 

For more information, please refer to the [wiki](https://github.com/UKNCSP/CMIP7-support/wiki).

If you would like access to one of the private repositories listed at [UKNCSP](https://github.com/UKNCSP) on GitHub, please open a "Request Access" issue within this CMIP7-support repository. Use the provided "Access Request Form" issue template and ensure you fill in all required fields, including your Name, Institution, and other relevant details.
